# 【記入サンプル】システムプロンプト設定

> ここから下は書き方のサンプルです。実際に書くときは上のテンプレに記入し、ここは削除してください。

## 概要

システムプロンプト（AI への指示文）を UI から変更・保存できる。

## 動作

1. 設定アイコンをクリックすると設定パネルが開く
2. テキストエリアに任意のシステムプロンプトを入力
3. 保存ボタンで永続化（localStorage）
4. 以降のチャットに反映される

## 受け入れ基準（AC）

- AC-1: 保存したシステムプロンプトはリロード後も残る
- AC-2: 既定値に戻すボタンで初期状態に戻せる
- AC-3: 空文字保存はバリデーションで阻止される

## データモデル

```
Settings
- systemPrompt: string
```

Settings は localStorage に永続化する。

## 非機能要件

- **セキュリティ**：保存値はクライアント内のみ。サーバーには送らない。
