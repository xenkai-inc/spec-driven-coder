## 7. モダンUIチェックリスト（仕上げ前に確認）

- [ ] アイコンは絵文字でなく SVG（Lucide / Heroicons）
- [ ] クリックできる要素に `cursor-pointer`
- [ ] ホバー時になめらかに変化（transition 150〜300ms）
- [ ] 文字と背景のコントラスト（本文 4.5:1 以上）
- [ ] キーボード操作で focus リングが見える
- [ ] 余白に一貫性がある
- [ ] スマホ表示で崩れない（375 / 768 / 1024px）
- [ ] prefers-reduced-motion を尊重
